# Portfolio Rules

## Position Lifecycle

```text
Research
 ↓
Watch
 ↓
Build Position
 ↓
Increase
 ↓
Hold
 ↓
Reduce
 ↓
Exit
 ↓
Review
```

## Capital Action Vocabulary

Use only:

- Research
- Observe
- Build
- Accumulate
- Hold
- Reduce
- Exit

Do not use:

- Buy
- Sell

Reason:

> Action must be tied to position state and capital allocation, not treated as an isolated order.

## Conviction

- Low
- Medium
- High
- Very High

## Priority

- S
- A
- B

## Review Frequency

- Daily
- Weekly
- Monthly
- Quarterly

## Traceability Rule

Every portfolio action must trace back to:

1. Living Database evidence.
2. Trading OS reasoning.
3. Portfolio position state.
4. Execution Log review when a trade occurs.
