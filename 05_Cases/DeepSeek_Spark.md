## Seven-Layer Reasoning

| Layer | Reading |
|---|---|
| Fact | DeepSeek Spark represents an efficiency-improving AI model or inference approach. |
| Physics | Lower inference cost can increase total compute, memory, and bandwidth usage through higher demand. |
| Engineering | Efficiency gains depend on model architecture, serving stack, KV cache, batching, and hardware utilization. |
| Economics | Jevons Paradox applies: lower unit cost can expand total consumption. |
| Finance | Beneficiaries depend on whether higher usage offsets lower cost per token and supports infrastructure spending. |
| Capital | Capital should not assume hardware demand falls; Memory and Equipment can still benefit if usage expands. |
| Trading | Treat efficiency news as a signal, not an automatic sell signal for infrastructure; wait for demand evidence. |
