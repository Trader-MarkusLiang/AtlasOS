# Case: Korea Memory CapEx

Expected Atlas output:

- Strengthens Memory.
- Even more strongly reinforces Equipment / Materials.
- Does not directly imply the cycle is ending.

## Seven-Layer Reasoning

| Layer | Reading |
|---|---|
| Fact | Korea Memory CapEx indicates continued investment in memory capacity or technology. |
| Physics | Memory remains a core AI bottleneck because HBM/DRAM capacity and bandwidth constrain AI systems. |
| Engineering | Memory expansion requires advanced equipment, materials, process control, and yield improvement. |
| Economics | CapEx can signal strong demand, but supply growth must be compared with pricing and utilization. |
| Finance | Equipment and Materials can benefit earlier through orders, while memory producers depend on cycle discipline. |
| Capital | Capital should strengthen Memory and even more strongly reinforce Equipment / Materials. |
| Trading | Do not assume capex means cycle end; confirm supply-demand balance and avoid chasing extended names. |
