# Case: Nomura FCF

Expected Atlas output:

- AI giants become more asset-heavy in the short term.
- Shovel sellers benefit.
- Track ROI / FCF continuously.

## Seven-Layer Reasoning

| Layer | Reading |
|---|---|
| Fact | Nomura FCF analysis points to AI giants becoming more capital-intensive in the short term. |
| Physics | AI infrastructure demand requires compute, memory, bandwidth, power, equipment, and materials. |
| Engineering | Scaling AI capacity requires data centers, accelerators, memory, networking, and fab supply chains. |
| Economics | Heavy capex is acceptable only if AI revenue, token growth, or enterprise ROI improves. |
| Finance | Free cash flow pressure is a warning signal for hyperscalers but can support shovel-seller revenue. |
| Capital | Capital should track ROI/FCF while favoring suppliers that benefit from sustained capex. |
| Trading | Stay constructive on shovel sellers, but reduce confidence if capex grows while ROI deteriorates. |
