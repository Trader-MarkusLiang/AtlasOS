# AI Bottleneck Index

Authority: classification system.

This file defines the bottleneck taxonomy. Current rankings belong in
`04_Current_State/Bottleneck_Map_v1.md`.

Atlas does not classify AI opportunities mainly by traditional industry. It classifies them by
AI-system bottleneck.

| Bottleneck | Content |
|---|---|
| Compute | GPU / ASIC |
| Memory | HBM / DRAM / NAND / SSD |
| Equipment | Etch / deposition / cleaning / CMP |
| Materials | Targets / precursors / CMP materials / hydrofluoric acid / quartz / wafers |
| Bandwidth | Optical modules / CPO / fiber / optical interconnect |
| Power | Power / liquid cooling / transformers / UPS |
| Workflow | Enterprise AI workflow |
| Industry AI | Industry AI / robotics / autonomous driving |
