# ROI Engine

Core principle:

> Capital Always Follows ROI.

## Key Metrics

| Metric | Meaning |
|---|---|
| AI CapEx | Are cloud providers still investing? |
| AI Revenue | Is investment converting into revenue? |
| FCF | Is free cash flow being consumed? |
| Token Growth | Is usage growing? |
| Inference Capacity | Is inference capacity still undersupplied? |
| Enterprise AI ROI | Are enterprises willing to keep paying? |

## Trading Interpretation

| Situation | Meaning |
|---|---|
| CapEx growth + ROI improvement | Bullish for shovel sellers |
| CapEx growth + ROI deterioration | Watch for valuation compression |
| CapEx slowdown + unclear ROI | Defensive stance |
| ROI improvement + valuation pullback | Add-position window |
