# Atlas Goal Evidence Index

This directory maps each Atlas goal to its current evidence, source reports, gaps, and next
evidence to collect.

Evidence indexes are governance artifacts. They do not create runtime behavior, cognition logic,
trading execution, broker connectivity, CDE authority, or prediction systems.

## Evidence Discipline

- `LIVE_PROVEN` requires actual provider or live path proof.
- `REAL_RUNTIME_PROVEN` requires daemon/UI/persistent runtime path proof.
- `CONTROLLED_FIXTURE_PROVEN` cannot be described as live.
- `ACCELERATED_ONLY` cannot be described as wall-clock stability.
- `PARTIAL` requires the missing evidence to remain visible.
- `EXTERNAL_BLOCKER` must preserve degraded behavior and never become fake freshness.
