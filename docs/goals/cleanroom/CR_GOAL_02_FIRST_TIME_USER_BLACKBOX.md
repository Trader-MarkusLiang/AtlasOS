# CR_GOAL_02 - FIRST-TIME USER BLACK-BOX

## Objective

Test Atlas strictly as a new user without inspecting implementation during the first pass.

## Required Journey

1. Open Atlas.
2. Understand what Atlas does.
3. Select Chinese.
4. Switch back to English.
5. Configure LLM.
6. Test provider.
7. Select model.
8. Add 3 assets without JSON.
9. Set percentages.
10. Save.
11. Start Atlas.
12. Confirm Running.
13. See first brief.
14. Open Markets.
15. See freshness or degraded state.
16. Open Portfolio.
17. See concentration.
18. Open Predictions.
19. Open Learning.
20. Ask Atlas a question.
21. Stop Atlas.

For each step record success, clicks, confusion, misleading status, hidden action, raw JSON/dict or
trace leakage, mixed language, and stale UI state.
