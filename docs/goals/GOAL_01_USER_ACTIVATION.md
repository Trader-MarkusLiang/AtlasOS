# GOAL 01 — ORDINARY USER ACTIVATION

## Objective

A first-time user must be able to configure and operate Atlas OS without:

- terminal
- JSON
- Python
- logs
- internal acronyms

## Required Journey

1. Open Atlas
2. Understand Atlas
3. Select language
4. Configure LLM
5. Test provider
6. Select model
7. Add assets
8. Set percentages
9. Start runtime
10. See first brief
11. Ask Atlas
12. Stop runtime

## Mandatory Fixes

- API key must truly submit
- provider test must use current form values
- asset setup must not require JSON
- runtime start must exist in user flow
- running state must be obvious
- Home must be Decision Brief-first
- no raw dict/JSON by default
- zh/en parity

## Product Pages

- Home
- Setup
- Ask Atlas
- Portfolio
- Markets
- Predictions
- Learning

## Acceptance

Complete only if the entire journey succeeds through real UI/runtime path.
