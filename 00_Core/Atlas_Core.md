# Atlas Core

Atlas OS is not a traditional software project. It is an AI-model-driven investment research
and trading decision knowledge system.

Its core value is the repeatable knowledge framework:

- Research framework
- Reasoning chain
- Capital migration logic
- AI bottleneck map
- Trading OS
- Trading discipline
- Sustainable knowledge-base iteration

Git exists here to:

1. Preserve Atlas core knowledge assets.
2. Support version management.
3. Let Codex, ChatGPT, and future agents read the same framework.
4. Avoid losing Atlas when it only exists inside chat context.

Current stage: move the Atlas core brain into Git before tool-building.
