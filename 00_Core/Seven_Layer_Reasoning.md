# Seven Layer Reasoning

Any news, opinion, chart, or research report must pass through this chain:

```text
L0 Fact
 ↓
L1 Physics
 ↓
L2 Engineering
 ↓
L3 Economics
 ↓
L4 Finance
 ↓
L5 Capital
 ↓
L6 Trading
```

The goal is not to explain news. The goal is to decide whether it changes:

1. Facts
2. Bottlenecks
3. ROI
4. Capital flows
5. Trading actions

## Layer Definitions

| Layer | Question |
|---|---|
| L0 Fact | What actually happened? |
| L1 Physics | Which physical bottleneck changed? |
| L2 Engineering | Can the solution be implemented at scale? |
| L3 Economics | Does the change improve unit economics or demand? |
| L4 Finance | Does it change revenue, margin, FCF, or CapEx? |
| L5 Capital | Where will capital flow next? |
| L6 Trading | What should be done with position, timing, and risk? |
